What can I do?

This is a browser based chat application. You can run around, visit worlds, chat with other people and play games. Enjoy your stay!





How do I move?

Good question! You can move around using the arrow keys on your keyboard. If nothing happens, try clicking the world to focus and try again. If the problem is still there, try reloading your page.





How do I move faster?

Glad you asked! You can modify your speed by clicking on the 'speed dial' button on the right hand side of your screen. There are 3 speed settings available:

- slow (blue dial) moves you one tile per click.

- medium (yellow dial) moves you two tiles per click.

- fast (red dial) moves you three tiles per click.

The speed cyles, so by clicking the button multiple times you return to the slow speed again.





How do I send a message?

At the bottom of your screen there is a big inputbox. By making sure the command is set on 'message', you can type your message and press ENTER. Your message will be send to everyone in the same world. Beware you can only send messages in a world and NOT in the lobby!





How can I see messages?

When someone sends a message, the messagebox will appear for a few seconds. If you want to see this again, press 't' on your keyboard. If you want to close the messagebox, you can press 'esc'.





What does the 'message' command do?

This will send a message to everyone in the same world as you.





What does the 'whisper' command do?

This will send a message to everyone in the same world that is close to you. The range of possible recipients is configurable. It's also possible to exclude certain clients by typing '/ostracize annoying_client'. For a complete description, type help using the 'whisper' command.





What does the 'play-game' command do?

This will start a game. For more info about the different games, type help (using the 'play-game' command).





What does the 'set history' command do?

This requires a number as input. This will make sure that the amount of messages in your messagebox is set to the given number.





What does the 'password' command do?

If you encounter a 'password' command, that means that the world you are trying to join is set to private. Which means that you need a password to enter. If you don't know the password u are screwed and will be returned to lobby!





What does the 'request admin' command do?

If you encounter a 'request admin' command, that means that the world you are joined to can be set to private. By entering a password in the inputbox (using the 'request admin' command) you become the admin of the world and everyone that want to try to enter the world will need this password. If an admin leaves, the world becomes public again.





How do I join a world?

Glad you made it to the lobby! If you want to join a world, you can move to a tile representing the different worlds. You can also click those tiles to join a world. If you are in a world, you can use the bridges to go to nearby worlds (or return to lobby).





How do I play a game?

Use the 'play game' command. This will start a game. For more info about the different games, type help (using the 'play-game' command).





What happens if I die?

I hope you won't need this information. But if you die in a world, everyone in that world will receive a message that you died.

You will see a different screen where you have 2 buttons. One will bring you back to your current world you died in.

The other one will bring you back to the lobby.





What do the coins do?

Try to collect as much coins as possible, they might give you some extra benefits! :)

Take a look at the achievement setting and try to figure out how it works.





How do I turn off the music?

You shouldn't! But if you insist, you can press the sound button on the right hand side of your screen.





How do I make a world private?

See the 'What does the 'request admin' command do?' question for more info. :)

What does bot do?

The bot has two commands: messenger and survey.

- sending a message starting with 'messenger', then the name of a world followed by a message, the bot will walk to the given world and deliver the message there. Try: 'messenger Grassworld Don't fall in the water!'

- sending a message starting with 'survey', a question and the possible answers, all separated from each other using a semicolon, will make the bot take a survey and send the results in the chat. Try: 'survey ; Which professor is cooler? ; Dirk Nuyens ; Bart Jacobs ; equally as cool' .
